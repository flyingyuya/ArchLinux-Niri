# ArchLinux Niri-VM Dotfiles

这是一个旨在分享个人在ArchLinux上使用Niri窗口管理器的系统配置和美化方案的项目。

## 项目介绍

此项目包含了我在ArchLinux物理机环境中为Niri合成器及其生态系统精心配置的各种点文件（dotfiles）。它旨在提供一个功能齐全、视觉统一且高度优化的桌面体验。

配置内容涵盖了以下关键组件：

*   **窗口管理器**: Niri
*   **Shell**: Zsh, Fish
*   **终端模拟器**: Kitty
*   **文本编辑器**: Vim, Lite-XL
*   **状态栏**: Waybar (实现两种样式，可随意切换)
*   **应用程序启动器**: Fuzzel
*   **通知**: Mako
*   **锁屏**: Swaylock
*   **壁纸管理**: Waypaper
*   **系统信息工具**: Fastfetch, Btop
*   **音频可视化**: Cava
*   **输入法**: Fcitx5
*   **主题和美化**: Matugen (用于生成颜色方案), GTK 3/4, Fontconfig
*   **其他工具**: Satty, Wlogout, MPV

## 适用场景

*   **快速部署**: 希望在ArchLinux上快速搭建一个基于Niri的桌面环境的用户。
*   **美化爱好者**: 寻求独特且统一的Linux桌面美化方案的用户。
*   **Niri用户**: 正在寻找Niri配置灵感或希望借鉴现有配置的用户。
